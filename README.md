# study-tracker

Study Tracker is a simple Python application that helps me track my daily study sessions. 
It measures how long each session lasts, keeps track of my current study streak, and stores the total time I have spent studying.

When the program is closed, it automatically sends a summary to my private Discord channel with:

- study session duration,
- current study streak,
- total study time.

The application stores its data in a local JSON file.
This project was created to help me stay motivated while working on my larger backend projects.
Before I start studying, I launch the tracker, and when I finish, I stop it. Receiving a summary on Discord makes it easy to monitor my progress and maintain consistency.
